GL.iNET Flint 2 (GL-MT6000) custom build with LuCi.

Repository for building pesa1234's GL.iNET - MT 6000 custom build https://github.com/pesa1234/MT6000_cust_build based on cjom's GL-MT6000 custom OpenWrt firmware builder https://github.com/cjom/gl-mt6000 with following customizations:

- Mbed TLS relaced with OpenSSL
- WiFi UCODE scripts
- Ad Block Fast
- Bridger
- DDNS
- DNS Crypt Proxy
- MWAN
- OpenSSH and SFTP with hardened config from cjom
- SQM
- TCP BBR
- UPnP
- Wireguard
- ZRAM
- Removed: dropbear, odhcpd and ppp
